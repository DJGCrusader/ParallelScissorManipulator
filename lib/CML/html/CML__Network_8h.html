Enumeration used to identify a network node state.

Enumerator
NODESTATE_INVALID 

Invalid node state.

NODESTATE_UNKNOWN 

Unknown state - the default state on node creation.

The state will be changed when communication with the node is established.

NODESTATE_GUARDERR 

On a node guarding or heartbeat timeout, the state will change to guard error.

Stopped state (aka init state for EtherCAT)
